Workshop 1 Activity

Imagine the image above is the canvas of a little diagram editor. The user has selected a region shown with the dotted orange outline. They then invoke a UI feature which zooms the selected objects to fit neatly into the canvas window. We are going to implement this functionality - or at least the code that implements the computation.

You will need to implement four functions in the main.js file: overlappingIntervals, rectanglesOverlappingBoundary, sumAreasOverlappingBoundary and computeBounds.

(OK, we don't actually need to sum the areas, but it's more practice with reduce.)
